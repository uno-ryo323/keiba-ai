# CLAUDE.md

## 目的
このリポジトリは過去に作成した競馬AIコードを整理・再生するためのもの

## 方針
- 既存ロジックの意味は変えない
- まず説明、その後に改善提案
- いきなり大きな変更はしない

## 優先事項
- 可読性向上
- 構造整理
- 再実行可能にする

## 禁止
- ロジックを勝手に変更しない
- 一度に広範囲を修正しない

## 作業手順
1. コードの役割を説明
2. 問題点を整理
3. 改善案を提示
4. 承認後に修正

## 記録
- 作業フェーズが完了したら PROGRESS.md に内容・所感を追記する

## Git
- 作業開始時にタスク用ブランチを切る（例: `feature/phase1-selenium-update`）
- 1タスク完了ごとに git commit & push する
- コミットメッセージは日本語でタスク内容を簡潔に記載する
- タスク完了後にプルリクエストを発行する
- マージはユーザーが承認してから行う
